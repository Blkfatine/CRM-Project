FAQ

Pour ajouter un nouveau lead, accédez à la section "Leads" de votre tableau de bord et cliquez sur le bouton "Ajouter un lead". Vous serez redirigé vers un formulaire où vous devrez entrer les informations nécessaires telles que le prénom, le nom, l'email, le téléphone, la source, le statut, et des notes éventuelles. Une fois le formulaire rempli, cliquez sur "Enregistrer" pour ajouter le lead à votre liste.
Avant de fixer un rendez-vous avec un lead, il est important de vérifier les disponibilités dans le calendrier intégré de l'application. Vous pouvez consulter le calendrier pour voir les créneaux horaires disponibles et éviter les conflits avec d'autres rendez-vous déjà programmés. Assurez-vous également de vérifier les disponibilités de l'utilisateur assigné au lead pour une planification harmonieuse.
Pour importer des leads, allez dans la section "Leads" de votre tableau de bord et cliquez sur "Importer". Vous pourrez alors télécharger un fichier CSV contenant les informations de vos leads.
Vous pouvez télécharger le guide d'utilisation de l'application depuis le tableau de bord. Rendez-vous dans la section "Guide de support" où vous trouverez un lien pour télécharger le guide complet. Ce document contient toutes les informations nécessaires pour vous familiariser avec les fonctionnalités de l'application et vous aider à résoudre les problèmes courants.
Pour mettre à jour les informations d'un lead, accédez à la section "Leads" dans votre tableau de bord et sélectionnez le lead que vous souhaitez modifier. Cliquez sur le bouton "Modifier" pour ouvrir le formulaire de mise à jour. Vous pouvez alors ajuster les informations comme le prénom, le nom, l'email, etc. Une fois les modifications apportées, cliquez sur "Enregistrer" pour appliquer les changements.
Pour contacter le support technique, rendez-vous dans la section "Guide de support" de votre tableau de bord et contacter nous à tout moment pour toute question ou problème.
